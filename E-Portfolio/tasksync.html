TaskSync

TaskSync is a web-based platform designed to facilitate assignment management for teachers and students. It provides a dedicated teacher and student login system, allowing for secure access and role-based functionalities. Teachers can upload assignments, track submission statuses, and verify student-submitted PDFs, ensuring an organized workflow.

Key Features

  • Secure teacher and student login
  • Role-based functionalities for assignment management
  • Ability for teachers to upload and track assignments
  • Student submission of PDFs

Technologies Used

  • HTML5
  • CSS3
  • JavaScript
